## Task List

Task List is a simple web application that allows you to create a list of tasks that you can store, filter, and delete. 

## Live Demo:
www.jazdosanjh.com/tasklist/

## Built With

* HTML5
* CSS3
* Materialize CSS
* JavaScript ES6
